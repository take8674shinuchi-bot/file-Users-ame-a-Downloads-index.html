かんたん広告ブロッカー

Chrome で広告とトラッカーをブロックするシンプルな拡張機能。
ワンクリックで ON/OFF。外部サーバーへの送信は一切なく、すべてローカルで完結します。

⬇ 拡張機能をダウンロード（ZIP） GitHub で見る

Chrome ウェブストア未配布のため、開発者モードで読み込んでご利用ください。

🛡️

ワンクリック ON/OFF

ツールバーのアイコンから切り替え。タブごとのブロック数も表示します。

🌐

EasyList 採用・約3万件

広告ドメインの通信を遮断し、さらに広告の枠自体を CSS で非表示にします。

🔒

外部送信ゼロ

収集も外部通信もなし。ブロック処理はすべて端末内で完結します。

インストール手順

  1. 上の「ダウンロード」から ZIP を取得し、展開します。
  2. Chrome で chrome://extensions を開き、右上の「デベロッパー モード」を ON にします。
  3. 「パッケージ化されていない拡張機能を読み込む」をクリックし、展開した kantan-ad-blocker フォルダを選びます。
